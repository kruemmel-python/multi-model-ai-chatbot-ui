Installationsdokumentation

Diese Dokumentation führt Sie durch den Installationsprozess aller benötigten Komponenten und Bibliotheken, um die Anwendung auszuführen. Bitte folgen Sie den Anweisungen Schritt für Schritt.

1. Voraussetzungen

  • Python 3.9 oder höher muss installiert sein.
  • Ein funktionierendes Internetverbindung.
  • Ein Texteditor oder eine IDE Ihrer Wahl.

2. Python Umgebung einrichten

2.1. Virtuelle Umgebung (Empfohlen)

Es wird empfohlen, eine virtuelle Umgebung zu verwenden, um Abhängigkeiten für dieses Projekt zu isolieren.

  1. Erstellen Sie eine virtuelle Umgebung: 
    python -m venv venv
  2. Aktivieren Sie die virtuelle Umgebung:
    • Windows: 
      .\venv\Scripts\activate
    • macOS/Linux: 
      source venv/bin/activate

3. Installation von Ollama

Ollama ist notwendig, um lokale Sprachmodelle auszuführen.

3.1. Ollama installieren

Besuchen Sie die Ollama Website und laden Sie die passende Version für Ihr Betriebssystem herunter. Befolgen Sie die Anweisungen auf der Webseite, um Ollama zu installieren. Nach der Installation können Sie Ollama über die Befehlszeile verwenden.

3.2. Ollama Modelle herunterladen

Nach der Installation von Ollama müssen die benötigten Modelle heruntergeladen werden. Eine Liste der verfügbaren Modelle finden Sie im config.py oder auf der offiziellen Ollama Website. Um ein Modell herunterzuladen, führen Sie folgenden Befehl in der Konsole aus:

ollama pull phi4:latest

Ersetzen Sie phi4:latest mit dem gewünschten Modell.

4. API Keys

4.1. Mistral API Key

Um die Mistral API zu verwenden, benötigen Sie einen API-Schlüssel.

  1. Besuchen Sie die Mistral API Keys Seite.
  2. Melden Sie sich an oder erstellen Sie ein Konto.
  3. Erstellen Sie einen neuen API-Schlüssel und kopieren Sie ihn.
Speichern Sie diesen API-Schlüssel in der .env Datei als MISTRAL_API_KEY.

4.2. Gemini API Key

Um die Google Gemini API zu verwenden, benötigen Sie einen API-Schlüssel.

  1. Besuchen Sie die Google AI Seite.
  2. Melden Sie sich an oder erstellen Sie ein Konto.
  3. Erstellen Sie einen neuen API-Schlüssel und kopieren Sie ihn.
Speichern Sie diesen API-Schlüssel in der .env Datei als GEMINI_API_KEY.

5. Python Bibliotheken installieren

Die folgenden Python-Bibliotheken werden benötigt. Führen Sie diesen Befehl aus, nachdem Sie die virtuelle Umgebung aktiviert haben:

pip install -r requirements.txt

Falls requirements.txt nicht vorhanden ist oder Sie die Bibliotheken manuell installieren möchten, installieren Sie diese einzeln mit pip install:

  • mistralai
  • google-generativeai
  • pydub
  • gradio
  • pygments
  • python-dotenv
  • black
  • openpyxl
  • python-docx
  • fpdf
  • python-pptx
  • transformers
  • torch
  • Pillow
  • playsound
  • gTTS
  • PyPDF2
  • loguru
  • requests

6. Konfiguration der .env Datei

Erstellen Sie eine Datei namens .env im Hauptverzeichnis des Projekts. Fügen Sie die API-Schlüssel wie folgt hinzu: 

MISTRAL_API_KEY=Ihr_Mistral_API_Schlüssel
GEMINI_API_KEY=Ihr_Gemini_API_Schlüssel
Ersetzen Sie Ihr_Mistral_API_Schlüssel und Ihr_Gemini_API_Schlüssel mit Ihren tatsächlichen API-Schlüsseln.

7. Anwendung starten

Um die Anwendung zu starten, führen Sie den folgenden Befehl im Hauptverzeichnis aus:

python gradio_interface.py

Dadurch wird die Gradio-Benutzeroberfläche in Ihrem Webbrowser geöffnet.

8. Optional: TTS Voice ändern

Die Standard TTS-Stimme ist auf Alex von Apple eingestellt. Um dies zu ändern, passen Sie den Wert von tts_voice im config.py an. Eine Liste mit den verfügbaren Stimmen können Sie via say -v ? in der Konsole abrufen.

9. Fehlerbehebung

  • Fehler beim Starten: Stellen Sie sicher, dass alle Bibliotheken installiert sind und die Umgebungsvariablen korrekt gesetzt sind. Überprüfen Sie die Konsolenausgabe auf Fehlermeldungen.
  • Ollama Modelle: Stellen Sie sicher, dass Sie die gewünschten Modelle heruntergeladen haben.
  • API Fehler: Stellen Sie sicher, dass Ihre API-Schlüssel korrekt sind.

Wenn Sie auf Probleme stoßen, konsultieren Sie die Dokumentation der einzelnen Bibliotheken oder suchen Sie nach Lösungen im Internet.
